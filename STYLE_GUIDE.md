# Punjabi (pa-IN) Python Docs Style Guide

## Never Translate

### Python Keywords

if
else
elif
for
while
class
def
return
import

### Variable Names

x
y
count
user_name
my_list

### RST Roles

:func:`print`
:class:`list`
:mod:`os`
:ref:`section`

### URLs

Leave all URLs unchanged.

---

## Code Blocks

Translate:

- Comments
- Explanatory text

Do not translate:

- Python keywords
- Variable names
- Function names
- Class names
- Module names
